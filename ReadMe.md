# LearnSeries-CSharp

A small collection of C# learning projects and examples designed to demonstrate core concepts and patterns.

## Contents
- Examples: console apps, LINQ samples, async/await demos
- Projects: beginner-to-intermediate exercises

## Getting Started
1. Open the solution in Visual Studio or VS Code with the C# extension.
2. Restore NuGet packages: `dotnet restore`.
3. Build and run a project: `dotnet run --project ./path/to/project`.
